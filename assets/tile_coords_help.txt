Как снять координаты тайла на листе (сетка 16×16 px)
===================================================

Формат для игры: Vector2i(КОЛОНКА, СТРОКА) — оба числа с НУЛЯ, как в Godot TileSet.

Способ 1 — Aseprite (удобно для тайлсетов)
-----------------------------------------
  • Открыть PNG → Sprite → Grid & Guide → установить Width=16, Height=16.
  • В статус-строке или при риовании видно положение курсора в пикселях.
  • column = floor(pixel_x / 16), row = floor(pixel_y / 16).

Способ 2 — GIMP / Photoshop / Photopea
---------------------------------------
  • Включить линейку (Rulers), смотреть координаты левого верхнего угла нужного квадрата.
  • column = floor(x / 16), row = floor(y / 16).

Способ 3 — только калькулятор
-----------------------------
  • Левый верхний угол нужного квадрата в пикселях: (Px, Py).
  • column = Px / 16 (целочисленно), row = Py / 16.

Способ 4 — Godot
----------------
  • Создать TileSet в редакторе, добавить атлас topDown_baseTiles.png, размер тайла 16×16.
  • В палитре тайлов при выборе ячейки смотреть координаты atlas.

Что прислать разработчику
-------------------------
  Например: «трава — колонка 4, строка 5» → в коде будет Vector2i(4, 5).
